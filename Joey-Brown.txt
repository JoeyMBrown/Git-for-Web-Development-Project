Stretch Questions

    1. What is the difference between an inline element and a block element?
      A block element starts on a new line and takes up the whole width of the page.  While an inline element does not start a new line and only takes up as much width is necessary to contain the content within the element.
    2. What happens when an element is positioned absolutely? 
      When you position an element absolutely it allows you to specify exactly where the element will be on the page using the values; top, left, bottom, and right.
    3. How do I make an element take up only the amount of space it needs but also have the ability to give it a width?

    5. In your own words, explain the box model. What is the "fix" for the box model, in other words, how do we make all elements respect the width we've given them? 
      The box model is a way of explaining how all HTML elements are viewed as boxes.  We can make all elements respect the width we've given them by using css to define the widths we want them to have.
